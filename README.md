# 100天健康蜕变打卡（PWA）

这是一个**渐进式网页应用（PWA）**，可以像手机 App 一样添加到桌面，支持离线使用、每日打卡、日历回顾、数据统计和每周复盘。

## 📁 文件说明

| 文件/文件夹 | 说明 |
|------------|------|
| `index.html` | 应用主页面 |
| `manifest.json` | PWA 配置文件（应用名称、图标、主题色等） |
| `sw.js` | Service Worker，支持离线缓存 |
| `icons/` | 应用图标（192×192 和 512×512） |
| `README.md` | 本说明文件 |

## 🚀 使用方法

### 方法一：部署到 GitHub Pages（推荐，可安装到手机桌面）

1. 在 GitHub 新建一个公开仓库，例如 `health100`。
2. 把这个文件夹里的所有文件上传到仓库根目录：
   - `index.html`
   - `manifest.json`
   - `sw.js`
   - `icons/` 文件夹
3. 进入仓库 **Settings → Pages → Source**，选择 `main` 分支，保存。
4. 等待几分钟后，访问 `https://你的用户名.github.io/health100`。
5. 浏览器会提示“安装应用”，或在菜单中选择“添加到主屏幕”。

### 方法二：本地使用（无法安装到桌面，但可打卡）

1. 直接双击 `index.html` 用浏览器打开。
2. 所有数据存在浏览器 localStorage 中。
3. **注意**：本地文件打开时，部分浏览器可能不支持 Service Worker 和安装提示。

### 方法三：本地服务器打开

如果你电脑上有 Python：

```bash
# 进入本文件夹
cd "100天健康打卡PWA"

# Python 3
python -m http.server 8080

# 然后浏览器访问 http://localhost:8080
```

或者用 Node.js：

```bash
npx serve .
```

## 📱 安装到手机桌面

### 安卓（Chrome）
1. 用 Chrome 打开部署后的网址。
2. 点击右下角或菜单中的“**添加到主屏幕**”。
3. 确认后，桌面会出现“健康打卡”图标。

### iPhone（Safari）
1. 用 Safari 打开部署后的网址。
2. 点击底部分享按钮 ⬆️。
3. 选择“**添加到主屏幕**”。
4. 确认后，桌面会出现图标。

### 电脑（Chrome/Edge）
1. 打开部署后的网址。
2. 地址栏右侧会出现安装图标，或点击菜单 → “安装 100天健康蜕变打卡”。

## 💾 数据备份

- 应用内“设置”页面支持**导出 JSON 数据**和**导入 JSON 数据**。
- 建议每周导出一次，防止浏览器数据被清除。
- 换手机或换浏览器时，可以通过导入 JSON 恢复数据。

## ⚠️ 重要提醒

本应用仅为生活记录工具，**不能替代医疗建议**。你有血糖高、重度脂肪肝、肾阴阳两虚等情况，开始执行前请务必咨询医生。

## 🔧 自定义图标

如果你想更换应用图标：
1. 准备 192×192 和 512×512 的 PNG 图片。
2. 替换 `icons/icon-192.png` 和 `icons/icon-512.png`。
3. 重新部署即可。

## 🛠️ 技术栈

- 纯 HTML + CSS + JavaScript
- 无外部依赖
- 数据存储：localStorage
- PWA：Web App Manifest + Service Worker
